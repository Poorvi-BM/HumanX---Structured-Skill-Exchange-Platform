HumanX – Structured Skill Exchange Platform

HumanX is a full-stack platform that enables intent-based collaboration, session-based learning, and trust-driven interactions. It combines structured communication with personal productivity tools to create a focused and meaningful user experience.

🌟 Features
	•	 Intent-based posts (Doubt, Project, Career)
	•	 Help offers & collaboration requests (no spam interactions)
	•	 Inbox system for managing responses
	•	 Session booking with lifecycle:
	•	REQUESTED → ACCEPTED → COMPLETED / CANCELLED
	•	 Chat unlocks only after session acceptance
	•	 Rating & review system after sessions
	•	 Dynamic trust score based on user behavior
	•	 Profile system with credibility insights
	•	 Calendar view for session tracking


🧩 Workspace Module
	•	🎯 Daily focus tracking
	•	🔥 Productivity streak system
	•	📌 Task management
	•	📝 Quick to-do notes


🛠️ Tech Stack
	•	Frontend: React.js
	•	Backend: Django + Django REST Framework
	•	Database: SQLite
	•	Authentication: JWT (SimpleJWT)
	•	API Testing: Postman
